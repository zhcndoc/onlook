# 管理员仪表板

Onlook 管理员仪表板是一个面向自托管部署的可选管理工具。

## 功能

- **用户管理** - 查看和管理用户账户、项目及访问权限
- **订阅管理** - 处理用户订阅和价格等级
- **限速管理** - 监控和调整 API 速率限制
- **部署跟踪** - 查看和管理用户部署
- **域名管理** - 管理自定义域名及验证状态
- **产品与定价** - 配置可用产品和价格等级
