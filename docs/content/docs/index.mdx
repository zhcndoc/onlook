# 欢迎来到 Onlook！

Onlook 是一款“设计师的光标”，使设计师能够直接在浏览器 DOM 中对 React 和 TailwindCSS 项目进行实时编辑。它实现了设计与开发之间的无缝集成。

![Onlook 界面](/images/onlook-interface.png)
*Onlook 直观的设计界面让你可视化编辑 React 组件*

## 什么是 Onlook？

Onlook 让你能够：

- 直接在类似浏览器的界面中编辑设计
- 更改自动反映到代码中
- 使用 AI 辅助生成和修改代码
- 通过主题系统管理项目样式
- 直接从应用部署网站

借助 Onlook，设计师能够掌控实施过程，缩小设计与开发之间的差距。无需深入编码知识，即可创建生产就绪的代码。
