## 常规问题

### 什么是 Onlook？

Onlook 是一个开源的“设计师鼠标指针”，使设计师能够直接在浏览器 DOM 中实时编辑 React 和 TailwindCSS 项目。它实现了设计与开发的无缝集成。

### Onlook 是免费使用的吗？

是的，Onlook 完全开源且免费使用。您可以从[GitHub](https://github.com/onlook-dev/onlook)克隆代码库并在本地运行。

### 我可以用 Onlook 构建什么类型的项目？

Onlook 适用于任何 React 和 TailwindCSS 项目。您可以构建网站、网络应用、仪表板等。

## 技术问题

### Onlook 使用了哪些技术？

Onlook 采用以下技术构建：
- Next.js
- Drizzle
- Supabase
- Bun

### 我可以用 Onlook 为我现有的项目服务吗？

可以，Onlook 适用于任何 Next.js + TailwindCSS 项目。您可以导入现有项目，立即开始编辑。

### Onlook 与类似工具相比如何？

Onlook 是 Bolt.new、Lovable、V0 和 Figma Make 等工具的开源替代品，拥有其独特的功能和设计到代码的工作流程方法。
